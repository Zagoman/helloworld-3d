What is Three.js?

"Three.js is a cross-browser JavaScript library and application programming interface (API) used to create and display animated 3D computer graphics in a web browser using WebGL."

https://en.wikipedia.org/wiki/Three.js

What does that even mean?

  • Cross-browser

    • Works in Multiple browsers

  • JavaScript Library

    • An useful amount of abstraction to make it easy to perform a task

  • Creating and animating 3D graphics in the browser using WebGL

    • To create 3D in the browser, a JavaScript API called WebGL must be used, and with Three.js, this process becomes easier.

Now we need to understand what a Renderer is

A renderer will capture the relationship between a Camera and a Scene

The renderer draws the the portion of 3D Objects present on the scene, that are visible through the camera "lens"

Still confusing... Let's put you in the story

Let's imagine you are a photographer, and you arrive to a beautiful landscape. You want to capture this landscape in a picture fomat.

In this scenario, the landscape is the Scene, and when you take an image, you are using your "camera" to take that image. The result of you pointing your camera to the scene and taking a photo, produces a Render of an image.

Therefore, the Renderer is what produces this image.

Let's Create a Renderer

After you add you import Three.js' library, select the canvas you created, then create the renderer passing to it where it should render to, which is the canvas element. 

                
                  import * as THREE from "https://cdn.jsdelivr.net/npm/three@0.118/build/three.module.js";
  
                  //Select your canvas
                  const canvas = document.querySelector("#myCanvas");
  
                  //Instanciating or Creating a new Renderer
                  const renderer = new THREE.WebGLRenderer({ canvas: canvas });

                  //Rendering the scene. This line should be the last one
                  renderer.render(scene, camera)

Ps.: The renderer needs an Object as a parameter.

Why can't we see anything?

Okay, so let's create a box geometry to add to the scene.

A box has 6 sides, but it contains 3 measurements. The width, height and depth.

To create the box, we also need its material, it can be as simple as a color 

                
                  /** 
                  * Previous code
                  */
  
                  //creating a box
                  const boxWidth = 1;
                  const boxHeight = 1;
                  const boxDepth = 1;
                  const geometry = new THREE.BoxGeometry(boxWidth, boxHeight, boxDepth);
                  const material = new THREE.MeshBasicMaterial({color: 0xff0000});
  
                  //Combine all items together
                  const box = new THREE.Mesh(geometry, material);
  
                  //Inserting our box into the Scene
                  scene.add(box)

I still can't see anything, shouldn't I see it by now?

The box now exists, but the problem is the camera.

Moving the camera around

to move the camera, we have to understand the X, Y and Z axis

https://threejs.org/manual/#en/fundamentals

we can move the camera in space, moving its position with its X, Y and Z coordinations 

                
                  /** 
                  * Previous code
                  */
  
                  //creating the camera
                  const fov = 75 //in degrees
                  const aspect = 2 // width / height
                  const near = 0.01 //how close to the lens
                  const far = 5 // how far from the lens
                  const camera = new THREE.PerspectiveCamera(fov, aspect, near, far) //the actual camera
  
                  camera.position.z = 3;
                  camera.position.x = 0;
                  camera.position.y = 0;

Now our box look like a square

We do the same that we did with the camera, we move the box in space. 

                
                  /** 
                  * Previous code
                  */
  
                  //Combine all items together
                  const box = new THREE.Mesh(geometry, material);
                  box.position.z = 0;
                  box.position.x = 2;
                  box.position.y = 0;
  
                  //Inserting our box into the Scene
                  scene.add(box)
